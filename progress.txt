## Running the Application

```bash
# Install dependencies
pnpm install

# Build shared package first
pnpm --filter @nonotion/shared build

# Start development servers
pnpm dev

# Or start individually
pnpm --filter @nonotion/api dev  # http://localhost:3001
pnpm --filter @nonotion/web dev  # http://localhost:5173
```

## Running Tests

```bash
# E2E tests (requires dev servers)
pnpm --filter @nonotion/e2e test:e2e

# E2E with UI
pnpm --filter @nonotion/e2e test:e2e:ui
```

---

## Architecture Notes

- Block registry pattern allows easy addition of new block types
- All entities have `version` field for future LWW sync
- Storage adapter interface allows swapping JSON for database
- Admin user auto-logged in (Phase 1 only)
